PyTriEvo (далее Программа) является клоном программы TriEvo (https://github.com/Djaler/Trievo) на Python'е.
Программа создана для экспериментов с генетическими алгоритмами.
В разработке программы используется паттерн MVC. В директории проекта Вы найдёте пакеты "Model", "View" и "Controller". В пакете "Model" присутствует модуль "Population", который реализует генетический алгоритм.
